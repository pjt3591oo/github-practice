# 기능정의서

## 계정관리

1. 지갑 가져오기(private key를 입력받아서 버튼을 누른다.)
2. 보유하고 있는 이더리움 잔액
3. NFT 토큰 수량

## NFT

1. 발행

  폼데이터
    1.1. 이미지 => IPFS
    1.2. 작성자
    1.3. 업로드 날짜(timestamp/ms => 문자열) -> 사용자로부터 입력받지 않음

2. NFT 토큰을 가지고 발행정보 조회

3. 이미지를 업로드 된 이미지 익스플로러 사이트 이동
